# Jann Inc

Company website for Jann Inc — a tech product studio.

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- shadcn/ui

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Pages

- `/` — Home landing page
- `/products` — Product showroom
- `/contact` — Contact form
